# Config Notes

The config files in this directory are examples from the paper experiments.
Before running them, replace all local placeholders with your own paths and
model endpoints:

- `<SDMC_ROOT>`
- `<SERVER1_DATA_ROOT>`
- `<SPIDER_ROOT>`
- `<BIRD_ROOT>`
- `<gpu-alias>`
- OpenAI-compatible endpoint URLs

Do not commit private API keys. Use environment variables or a private key file
outside the repository.
